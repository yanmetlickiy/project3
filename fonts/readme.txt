1) Находим по названию шрифта в гугле наш шрифт. Я делал поисковой запрос как "Swagger TTF". Первая ссылка, где был найден шрифт:
https://www.swagger.kr/font.html

2) Скачиваем файлы шрифтов
3) Все файлы оказались с расширением  ttf, но для поддержки шрифтов браузерами требуется файл с расширением .woff. 
4) Для решения этой проблемы я конвертировал шрифт из формата ttf в формат woff. В данном конвертере все удобно делается :https://convertio.co/ru/ttf-woff/
5) Скачиваем файл с расширеним .woff и помещаем в нашу папку fonts. И в файле fonts.css прописываем соединения файла шрифта с css.
